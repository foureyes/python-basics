Intro to Python

Fact

{% highlight pycon %} >>> def fact(n): ... if n == 0: ... return 1 ... else: return n * fact(n - 1) ... >>> fact(1) 1 >>> fact(0) 1 >>> fact(3) 6 {% endhighlight %}

Modules

{% highlight python %} >>> import m >>> m.hello_module() hi, i'm in yr modulez >>> import m as my_crazy_module_name >>> my_crazy_module_name.hello_module() hi, i'm in yr modulez >>> from m import hello_module >>> hello_module() hi, i'm in yr modulez {% endhighlight %}

PYTHON_PATH

{% highlight python %} >>> import sys >>> sys.path >>> sys.modules {% endhighlight %}

Running a module

{% highlight python %} if __name__ == “__main__”: do_your_stuff() {% endhighlight %}

Packages

  • A directory containing __init__.py
  • Code can go in __init__.py
  • Other modules can be imported under the package directory
{% highlight python %} import package_name import package_name.module_name from package_name import module_name {% endhighlight %}
