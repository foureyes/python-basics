Really about Python

  • Dynamic / interpreted programming language
  • Dynamically typed and type inference
  • Strongly typed
  • Whitespace significant
  • Allows imperetive, object oriented, and to some extent, functional programming.
  • Comprehensive standard library

These features lead to:

  • Ease of use
  • "One obvious way to do things" (unambiguous syntax)
  • Readability
  • Powerful
  • "Batteries included"
  • Ease of use - no need to wait for compile, no need to declare types, type checking at runtime
  • One way - No coercian like PHP or JavaScript
  • Readability - forces proper indentation; whitespace is both human and computer "readable"
